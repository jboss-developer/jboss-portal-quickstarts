Portal Quickstarts

The quickstarts demonstrate several ways how to create Portlets for Red Hat JBoss Portal. They provide small, specific, working examples that can be used as a reference for your own project.

Compatibility Information

This release of Portal Quickstarts is compatible with Red Hat JBoss Portal 6.2 running on JBoss EAP.

Available Quickstarts

  • Simplest Hello World Portlet
  • JSF2 Hello World Portlet
  • JSF2+RF4 Hello World Portlet
  • Navigation API Portlet
  • Page Composition API Portlet
  • CDI Generic Portlet
  • CDI Portlet with JSF
  • Portlet Using Portet-Specific CDI Scopes
  • Social Portlets
  • Portal Extension
  • Sample Portal
